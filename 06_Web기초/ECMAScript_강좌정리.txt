4. 논리연산자

   1)   값1  &&  값2  

       값1이 true이면 값2가 선택되고
       값1이 false이면 값1이 선택됨.

       ex.  10 && "홍길동" => "홍길동"
            
	    NaN && 100   => NaN

   2)  값1  ||  값2  

      값1이 true이면 값1이 선택되고
      값1이 false이면 값2이 선택됨.

        ex.  10 || "홍길동" =>  10
            
	    NaN || 100   =>   100


5. 객체 종류 3가지

   - 자바의 클래스와 동일하게 속성(변수)과 메서드(함수)제공함.

  1) 데이터 관련 객체
     -문자관련: String 
     -수치관련: Number
     -날짜관련: Date
     -배열: Array
     ..

  2) 브라우저 관련 객체 ( Browser Object Model: BOM )
    
     -Window객체: 웹브라우저 창 관련 객체 (<== window 이용해서 참조함 )
                 전역객체(************************)
	ex. Window window = new Window();
     -Screen객체: 웹브라우저 화면 관련 객체 (<== screen 이용해서 참조함 )
     -History객체: 웹브라우저 뒤로/앞으로 관련 객체 (<== history 이용해서 참조함 )
     -Location객체: 웹브라우저 URL 관련 객체  (<== location 이용해서 참조함 )
     -Navigator객체: 웹브라우저 자신 관련 객체 (<== navigator 이용해서 참조함 )

     ==> 웹브라우저를 open하면 자동으로 생성됨

      screen,history,location,navigator 는 Window객체의 속성으로 제공함.

      원래는 window.screen
          window.history 형식으로 써야 되지만, window는 전역객체이기 때문에 생략해서 사용이 가능하다.


   3) html 문서 관련 객체 ( Document Object Model : DOM )
    
     - Document 객체  (<== document 이용해서 참조함 )

     ==> 웹브라우저에서 임의의 html파일이 보여질 때 자동으로 생성됨.

     원래는 window.document 형식으로 써야 되지만, window는 전역객체이기 때문에 생략해서 사용이 가능하다.
