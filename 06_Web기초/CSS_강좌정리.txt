CSS_강좌정리

1. poistion 

 - poistion 이전 문법으로는 블럭레벨/인라인레밸 정도 배치가 가능함.
 - poistion + top/right/bottom/left 이용하면 원하는 위치에 배치 가능

2. poistion 종류

 1)  position:static

   - 기본방법
   - 블럭레벨은 세로배치,
     인라인레벨을 가로배치.
   - top/right/bottom/left 지정해도 적용이 안됨. (***********)
     역으로 static이 아닌 다른값을 사용해야 top/right/bottom/left 적용된다.

  2) position:relative
    
          -top/right/bottom/left 적용됨
	  -기준은 기본적으로 배치되는 것에 대한 기준임 ( position:static 배치가 기준 )

  3) position:absolute
   
         -top/right/bottom/left 적용됨
	 - 기준은 2가지 중 하나

	    가.  부모기준
                - 부모가 position:relative 지정했을 때

		  ex
		     <div> <== position:relative
		        <p> <== position:absolute




	    나. 화면(view port)이 기준
               - 부모가 position 지정하지 않았을 때

                  ex
		     <div> 
		        <p> <== position:absolute
		
		 

  4) position:fixed

        -top/right/bottom/left 적용됨
	-기준은 화면(view port)이 기준이됨.

글꼴(font)
 
 - 글꼴 카테고리 표현: Generic Family

    ex.      serif : 꼬리가 있음
        sans-serif : 꼬리가 없음
	    ..

- 글꼴 카테고리에 해당되는 실제 글꼴:  font-family

    serif : 타임즈글꼴, Geogia, ...
    sans-serif : consolas, ....

  
  글꼴설정방법

    font-family: font-family, font-family, ...., Generic Family

     font-family: 글꼴1, 글꼴2, .....,serif
     font-family: 글꼴1, 글꼴2, .....,sans-serif
